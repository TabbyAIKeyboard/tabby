title: Keyboard Shortcuts
description: Master Tabby with global hotkeys.
---

Keyboard shortcuts are the primary way to interact with Tabby. All shortcuts are global, meaning they work regardless of which application is in focus.

## Global Shortcuts

| Shortcut | Action |
| :--- | :--- |
| `Ctrl + \` | Open/Close Action Menu |
| `Ctrl + Alt + G` | Get AI Suggestion |
| `Ctrl + Shift + B` | Toggle Brain Panel |
| `Ctrl + Alt + I` | Toggle Interview Ghost Text |
| `Ctrl + Alt + J` | Launch Voice Agent |
| `Ctrl + Shift + X` | Immediately Stop Autotyping |

## Interview Copilot Shortcuts

These shortcuts are active when the Copilot panel or context is relevant.

| Shortcut | Action |
| :--- | :--- |
| `Alt + X` | Capture Screen & Analyze Problem |
| `Alt + Shift + X` | Update Analysis with New Context |
| `Alt + N` | Get Code Suggestions |
| `Ctrl + 1-6` | Switch between Copilot Tabs |
| `Ctrl + N` | Start a New Conversation Session |

## Navigation & Output

| Shortcut | Action |
| :--- | :--- |
| `Shift + Esc` | Accept suggestion |
| `Enter` | Paste AI result |
| `Esc` | Close the current panel or discard suggestion |
| `Tab` | Switch focus/modes within the Action Menu |
| `Ctrl + Arrows` | Move the floating UI panels around the screen |

<Callout type="info">
  You can customize some of these shortcuts in the **Settings** panel accessible from the system tray icon.
</Callout>
