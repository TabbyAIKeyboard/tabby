## What is Tabby?

Tabby is a **system-wide AI assistant** that lives at the point of input. Instead of switching between apps for AI help, it provides seamless integration through global shortcuts, ghost text suggestions, and desktop automation.

<Callout type="info">Tabby runs locally as a desktop application with Electron, connecting to powerful backend services for memory and AI processing.</Callout>
